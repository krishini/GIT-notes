Como activar git en un proyecto

Voy a la consola, me pongo dentro de la carpeta que quiero agregar a git

git init > inicializa un repositorio git en mi carpeta

en VS code cuando voy haciendo cambios me los muestra en la zona "changes"

git status > para ver status

git add > listar archivos que quiero pasar a staging

git commit -m "Mensaje concreto con mi commit concreto"

git push para pushear los cambios al repo

git checkout

con git checkout hash number podes detachear osea deshacer el cambio del commit que hayas elegido.

como crear un nuevo repositorio, o agregar el repositorio al cual voy a estar pusheando

git remote add origin https://github.com/krishini/GIT-notes (la direccion de mi repositorio)

para ver q repositorios tengo:

git remote -v

con fetch me fijo si hubieron cambios en mi repo

git branch -M main: para RENOMBRAR MI RAMA

git branch: para ver que ramas tengo

git push -u origin main: el -u es upstream. sabe que va a escribir en origin (el repo remoto) y en la rama q defini (main)

git pull origin main: traerme los cambios
